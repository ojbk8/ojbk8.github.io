# 离线下载保存`crx`插件的方法

打开[扩展程序](chrome://extensions/)开启开发者模式，复制的扩展程序id到下面的网站下载。

1. [Chrome Extension Downloader](https://chrome-extension-downloader.com/)
2. http://yurl.sinaapp.com/crx.php

例如：

* [油猴脚本](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)Tampermonkey的ID是`dhdgffkkebhmkfjojejmpbldmpobfkfo`
* 二维码(QR码)生成器(QR Code Generator)的ID是`pflgjjogbmmcmfhfcnlohagkablhbpmg`

# 离线安装`crx`插件

新版chrome必须开启开发者模式，把后缀为.crx的插件直接拖放安装
